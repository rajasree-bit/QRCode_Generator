# QR Code Generator (Python)

A simple and beginner-friendly Python project that creates a QR code for any URL entered by the user. The generated QR code is saved automatically as an image file on your system.

## Features

    -Accepts any URL as input
    -Generates a clean QR code image
    -Saves the QR code in PNG format
    -Lightweight and very easy to use
    -Requires only one external library (qrcode)

## Installation
Install the required package using pip:

pip install qrcode[pil]

## ▶️ How to Use
Download or clone this repository.

Open a terminal in the project folder.

Run the script:

python generate_qr.py


Enter the URL when asked.

The QR code will be saved automatically at:

C:\Users\Rajasree\OneDrive\Desktop\projects\qrcode1.png


Scan the generated image using any QR scanner app.

## Output

The program generates:
`qrcode1.png` — a scannable QR code image for the URL you entered.
